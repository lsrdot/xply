# Plano de Testes de Software

| CT     | Requisito | Descrição do Teste                                                                                                         | Etapas do Teste                                                                                                         | Resultado Esperado                                                                         |
|--------|-----------|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| CT-001 | RF-001    | Será verificado se o sistema permite o registro integral de novos usuários conforme os campos dispostos para preenchimento | 1. Acessar a landing page <br>2. Acessar a página de cadastro <br>3. Preencher e confirmar os dados cadastrais          | Formulários devem aceitar apenas dados válidos e indicar campos obrigatórios               |
| CT-002 | RF-002    | Verificar a funcionalidade de edição de perfil                                                                             | 1. Acessar o perfil do usuário cadastrado <br>2. Realizar alterações <br>3. Avaliar persistência das alterações         | Campos editados devem exibir as alterações corretamente após validação                     |
| CT-003 | RF-003    | Verificar funcionalidade de login                                                                                          | 1. Acessar a landing page <br>2. Preencher dados de login <br>3. Inserir credenciais válidas <br>4. Submeter formulário | Sistema deve autenticar credenciais e bloquear acessos inválidos                           |
| CT-004 | RF-004    | Realizar recuperação de senha de usuário previamente cadastrado                                                            | 1. Acessar a landing page <br>2. Clicar em "Esqueceu sua senha?" <br>3. Inserir dados válidos                           | Sistema deve aceitar apenas dados válidos e autenticados (e-mail cadastrado)               |
| CT-005 | RF-005    | Validar visualização e gestão de tarefas                                                                                   | 1. Acessar lista de tarefas <br>2. Selecionar tarefa específica <br>3. Iniciar modo Pomodoro                            | Tarefas devem ser exibidas de forma organizada; timer Pomodoro deve funcionar corretamente |
| CT-006 | RF-006    | Verificar a conclusão de tarefas pelo usuário                                                                              | 1. Selecionar tarefa ativa <br>2. Acionar opção "Concluir"                                                              | Status da tarefa deve ser atualizado imediatamente conforme o tempo de foco                |
| CT-007 | RF-007    | Verificar a visualização do mapa de conquistas                                                                             | 1. Acessar seção "Meus Troféus" <br>2. Navegar pelo mapa <br>3. Visualizar conquistas alcançadas                        | Troféus conquistados e disponíveis devem ser exibidos de forma clara e intuitiva           |
| CT-008 | RF-008    | Validar exibição da barra de XP e nível do usuário                                                                         | 1. Acessar a dashboard principal <br>2. Localizar indicador de progresso <br>3. Verificar atualização pós-tarefa        | Progresso (barra %) deve ser exibido e atualizado em tempo real após ganho de XP           |
| CT-009 | RF-009    | Verificar o progresso em relação às conquistas do usuário                                                                  | 1. Acessar página de relatório de conquistas <br>2. Emitir relatório                                                    | Métricas de produtividade/conquistas devem ser exibidas de forma clara e concisa           |
| CT-010 | RF-010    | Observar o mapa de conquistas/troféus                                                                                      | 1. Acessar página de conquistas <br>2. Checar se todas foram registradas conforme métricas                              | Todas as conquistas devem ser exibidas de forma fiel às métricas de desbloqueio            |
