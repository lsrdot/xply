# XPly

`Análise e Desenvolvimento de Sistemas`

`Desenvolvimento de uma aplicação móvel em um ambiente de negócio`

`3º Semestre`

</div>

O XPly é um aplicativo mobile que incentiva a produtividade de forma envolvente e interativa. Criado para ajudar os usuários a organizarem melhor seu tempo, ele transforma atividades diárias em uma experiência gamificada, tornando o processo mais motivador e dinâmico. Com um sistema de progressão e conquistas, o XPly estimula o equilíbrio entre diferentes áreas da vida, incentivando o foco e a constância de maneira leve e intuitiva.
